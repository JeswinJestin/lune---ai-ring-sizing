Place assets here. Vite serves files in `public` from the site root.

Folder structure:
- /ar/bands/{id}.png           Ring band images (transparent PNG)
- /ar/gems/{id}.png            Gemstone overlays (transparent PNG)
- /recommendations/{id}.jpg    Recommendation tile images
- /overlays/roi.png            Optional ROI guides

Naming conventions:
- Use numeric IDs matching catalog entries (1..9, etc.)
- Transparent backgrounds preferred
- Keep similar center alignment for band/gem pairs
